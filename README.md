# Sh-cho Tap

## How do I install these formulae?

`brew install sh-cho/tap/<formula>`

Or `brew tap sh-cho/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "sh-cho/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
